prompt.txt 파일의 내용을 확인했습니다. 뉴스 기사와 실적 발표를 각각 다른 형식으로
  요약하고 가공하기 위한 매우 구체적인 지침이 담겨 있네요. 이 지침을 바탕으로 봇
  개발 계획을 세웠습니다.

  개발 계획

  봇은 크게 세 부분으로 나뉩니다.
   1. 데이터 수집 (Scraper): X.com 'wall st engine' 페이지에서 새로운 게시물을
      가져옵니다.
   2. 콘텐츠 가공 (Processor): 가져온 게시물 내용을 prompt.txt의 지침에 따라 Gemini
      API를 이용해 가공합니다.
   3. 자동 포스팅 (Poster): 가공된 콘텐츠를 DCinside 나스닥 갤러리에 게시합니다.

  ---

  1단계: 프로젝트 환경 설정

   * 사용 언어: Python
   * 필수 라이브러리:
       * selenium & webdriver-manager: 웹 브라우저 자동화를 통해 X.com 및 DCinside와
         상호작용합니다.
       * google-generativeai: Gemini API를 호출하여 텍스트를 가공합니다.
       * python-dotenv: DCinside 로그인 정보, Gemini API 키 등 민감한 정보를
         안전하게 관리합니다.
   * 프로젝트 구조:
       * main.py: 봇의 전체 흐름을 제어하는 메인 스크립트.
       * scraper.py: X.com에서 데이터를 수집하는 모듈.
       * processor.py: Gemini API로 콘텐츠를 가공하는 모듈.
       * poster.py: DCinside에 게시물을 등록하는 모듈.
       * .env: API 키, 로그인 정보 등 민감한 데이터를 저장하는 파일 (보안을 위해 Git
         등 버전 관리에서 제외).
       * prompt.txt: 이미 작성된 프롬프트 파일.
       * requirements.txt: 프로젝트에 필요한 라이브러리 목록.

  2단계: 데이터 수집 (Scraper) 개발

   * Selenium을 사용하여 웹 브라우저를 열고 'wall st engine' X.com 페이지에
     접속합니다.
   * 가장 최신 게시물의 텍스트 내용을 가져옵니다.
   * 중복 게시를 방지하기 위해, 마지막으로 처리한 게시물의 고유 ID를 파일에
     저장하고, 다음 실행 시 해당 ID 이후의 새 게시물만 처리하도록 구현합니다.

  3단계: 콘텐츠 가공 (Processor) 개발

   * 수집한 게시물 텍스트가 '뉴스'인지 '실적 발표'인지 간단한 키워드(예: earnings,
     revenue, EPS 등)로 구분합니다.
   * prompt.txt에서 해당 유형에 맞는 프롬프트(뉴스용 또는 실적용)를 읽어옵니다.
   * 게시물 원문과 프롬프트를 조합하여 Gemini API에 전달하고, 가공된 텍스트를 결과로
     받습니다.

  4단계: 자동 포스팅 (Poster) 개발

   * 보안: DCinside 아이디와 비밀번호는 .env 파일에 저장하여 코드에 직접 노출되지
     않도록 합니다.
   * Selenium을 사용하여 DCinside에 로그인합니다.
   * 나스닥 갤러리의 글쓰기 페이지로 이동합니다.
   * Gemini API가 생성한 결과물에서 제목과 본문을 분리하여 각각의 입력창에 채워
     넣습니다.
   * '등록' 버튼을 클릭하여 게시를 완료합니다.

  5단계: 통합 및 스케줄링

   * main.py에서 위 세 가지 모듈(Scraper, Processor, Poster)을 순서대로 호출하여
     전체 프로세스를 자동화합니다.
   * 일정 시간(예: 10분)마다 이 프로세스를 반복하도록 스케줄링 기능을 추가합니다.
   * 각 단계에서 발생할 수 있는 오류(네트워크 문제, 페이지 구조 변경 등)에 대비한
     예외 처리 로직을 추가하여 봇이 안정적으로 동작하도록 만듭니다.
